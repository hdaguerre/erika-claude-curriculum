1 What does the full GTM pipeline do, in one sentence? ¿Qué hace el pipeline GTM completo, en una oración?
Takes a raw CSV of leads, cleans and enriches it with web data, scores each lead against ICP criteria, and pushes qualified leads to Apollo — turning a messy spreadsheet into a prioritized, ready-to-outreach contact list.
Toma un CSV crudo de leads, lo limpia y enriquece con datos web, puntúa cada lead según criterios ICP, y envía los leads calificados a Apollo — convirtiendo una planilla desordenada en una lista de contactos priorizada y lista para el outreach.
2 What is the WAT framework and what layer did Erika just build? ¿Qué es el framework WAT y qué capa construyó Erika?
WAT = Workflows, Agents, Tools. Erika built the Tools layer — Python scripts that do the actual work — and the Workflow layer — gtm_pipeline.md, the SOP that any agent can follow. She is the Agent: the decision-maker who reads the workflow and orchestrates the tools.
WAT = Workflows, Agents, Tools. Erika construyó la capa de Tools — scripts Python que hacen el trabajo real — y la capa de Workflow — gtm_pipeline.md, el SOP que cualquier agente puede seguir. Ella es el Agent: la tomadora de decisiones que lee el workflow y orquesta las herramientas.
3 What tool finds tech stack data for a target domain? ¿Qué herramienta encuentra datos de tech stack para un dominio objetivo?
tools/fetch_tech_stack.py — it calls the BuiltWith Actor on Apify, passing the domain, and returns the technology categories that company uses on their website. This feeds directly into the ICP scoring model.
tools/fetch_tech_stack.py — llama al Actor BuiltWith en Apify, pasando el dominio, y devuelve las categorías de tecnología que usa esa empresa en su sitio web. Esto alimenta directamente al modelo de puntuación ICP.
4 What's the difference between enrichment and scoring? ¿Cuál es la diferencia entre enriquecimiento y puntuación?
Enrichment adds data columns — what tools a company uses, whether they're hiring, how many employees they have. Scoring uses those columns to rank leads by ICP fit — converting raw data into a priority order. Enrichment asks "what is this company like?" Scoring asks "how good a fit is it?"
El enriquecimiento agrega columnas de datos — qué herramientas usa una empresa, si está contratando, cuántos empleados tiene. La puntuación usa esas columnas para rankear leads por ajuste ICP — convirtiendo datos crudos en un orden de prioridad. El enriquecimiento pregunta "¿cómo es esta empresa?" La puntuación pregunta "¿qué tan buen ajuste es?"
