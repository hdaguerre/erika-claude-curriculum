1. What pandas is and why it matters for lead work1. Qué es pandas y por qué importa para el trabajo con leads

pandas is a Python library for working with tabular data. Think of it as an in-memory Oracle table that you can query, filter, transform, and export — all with Python code. No database connection required.

A pandas DataFrame is exactly a SQL result set: rows and columns, each column has a name and a data type. When you load a CSV into a DataFrame, you have the full table in memory and can start working with it immediately.

Here are the direct SQL translations:

  • pd.read_csv("leads.csv") = SELECT * FROM leads — load the whole table
  • df.dropna(subset=["email"]) = WHERE email IS NOT NULL — filter out empty values
  • df.drop_duplicates() = SELECT DISTINCT * — remove exact duplicate rows
  • df[df["email"].str.contains("@")] = WHERE email LIKE '%@%' — filter rows by a condition
  • df.to_csv("output.csv") = exporting the result set to a file

2. The lead cleaning pipeline: 5 operations every list needs2. El pipeline de limpieza de leads: 5 operaciones que toda lista necesita

Before any lead list is usable, it needs to go through a standard cleaning sequence. This is the first real tool you'll own:

  1. Load the file — df = pd.read_csv("leads.csv")
  2. Inspect shape — print(df.shape) tells you rows × columns. df.head() shows the first 5 rows. Always look before you clean.
  3. Drop rows with missing emails — df = df.dropna(subset=["email"])
  4. Drop duplicates — df = df.drop_duplicates()
  5. Export the cleaned file — df.to_csv("cleaned_leads.csv", index=False)

After steps 3 and 4, print a count: print(f"Removed {original_count - len(df)} rows"). Always know what your cleaning removed.

Antes de que cualquier lista de leads sea usable, necesita pasar por una secuencia de limpieza estándar. Esta es la primera herramienta real que vas a tener:

  1. Cargar el archivo — df = pd.read_csv("leads.csv")
  2. Inspeccionar la forma — print(df.shape) te dice filas × columnas. df.head() muestra las primeras 5 filas. Siempre mirá antes de limpiar.
  3. Eliminar filas con emails vacíos — df = df.dropna(subset=["email"])
  4. Eliminar duplicados — df = df.drop_duplicates()
  5. Exportar el archivo limpio — df.to_csv("cleaned_leads.csv", index=False)

Después de los pasos 3 y 4, imprimí un conteo: print(f"Se eliminaron {original_count - len(df)} filas"). Siempre sabé qué eliminó tu limpieza.
