ConceptsConceptos

1. Connecting MCP + image generation into a pipeline1. Conectando MCP + generación de imágenes en un pipeline

Each tool built separately this week — Figma MCP reads design data, image gen creates visuals, Python scripts manage files. Today they connect.

The pipeline:

  1. Figma MCP reads brand frame colors and fonts
  2. Claude constructs an image generation prompt using those exact values
  3. generate_image.py creates the asset and saves it to brand-kit/assets/

This is the same orchestration pattern as the GTM pipeline — Claude reads from one source, transforms the data, calls a tool with the result.

Cada herramienta construida por separado esta semana — Figma MCP lee datos de diseño, la generación de imágenes crea visuales, los scripts de Python gestionan archivos. Hoy se conectan.

El pipeline:

  1. Figma MCP lee los colores y fuentes del frame de marca
  2. Claude construye un prompt de generación de imágenes usando esos valores exactos
  3. generate_image.py crea el asset y lo guarda en brand-kit/assets/

Este es el mismo patrón de orquestación que el pipeline GTM — Claude lee de una fuente, transforma los datos, llama a una herramienta con el resultado.

2. Directing Claude Code as a creative director2. Dirigir Claude Code como directora creativa

You don't generate images manually — you write creative briefs in natural language, and Claude translates them into technical API calls. "I want a LinkedIn banner that feels warm and professional, uses terracotta and cream, and shows abstract flowing shapes" → Claude writes the exact prompt for the image API.

No generás imágenes manualmente — escribís briefs creativos en lenguaje natural y Claude los traduce en llamadas técnicas a la API. "Quiero un banner de LinkedIn que se sienta cálido y profesional, use terracota y crema, y muestre formas abstractas fluidas" → Claude escribe el prompt exacto para la API de imágenes.

Key Insight

This is the pattern for all creative work with Claude Code: you define the aesthetic direction in plain language, Claude handles the technical translation. You're the art director — Claude is the production team.

Exercise (~90 minutes)Ejercicio (~90 minutos)
  1. Ask Claude Code to read your brand-kit/CLAUDE.md and extract the color palette and visual style description. Ask Claude to summarize them in one paragraph.
    Pedile a Claude Code que lea tu brand-kit/CLAUDE.md y extraiga la paleta de colores y la descripción del estilo visual. Pedile a Claude que los resuma en un párrafo.
  2. Ask Claude: "Using the visual style from my CLAUDE.md, write 3 different image generation prompts — one for a LinkedIn banner (1584x396), one for a profile background, one for a post visual. Make the prompts technically specific: include exact color descriptions, style, composition, and dimensions."
    Preguntale a Claude: "Usando el estilo visual de mi CLAUDE.md, escribí 3 prompts de generación de imágenes distintos — uno para un banner de LinkedIn (1584x396), uno para un fondo de perfil, uno para un visual de post. Hacé los prompts técnicamente específicos: incluí descripciones exactas de colores, estilo, composición y dimensiones."
  3. Run all 3 prompts through tools/generate_image.py. You should have 3 output files.
    Ejecutá los 3 prompts a través de tools/generate_image.py. Deberías tener 3 archivos de output.
  4. Review the results — open each image. If any feel off, tell Claude what's wrong and ask it to refine the prompt. Re-run until you have versions you're genuinely happy with. Iteration is normal.
    Revisá los resultados — abrí cada imagen. Si alguna no te convence, decile a Claude qué está mal y pedile que refine el prompt. Volvé a ejecutar hasta que tengas versiones con las que estés genuinamente conforme. La iteración es normal.
  5. Save the final images to brand-kit/assets/ with clear names: linkedin-banner.png, profile-bg.png, post-template.png.
    Guardá las imágenes finales en brand-kit/assets/ con nombres claros: linkedin-banner.png, profile-bg.png, post-template.png.
  6. Go live: upload the LinkedIn banner to your actual LinkedIn profile. This is the first visual piece of your brand kit in the world.
    Publicalo: subí el banner de LinkedIn a tu perfil real de LinkedIn. Esta es la primera pieza visual de tu brand kit en el mundo.
