1. What the Figma MCP server can do1. Qué puede hacer el servidor Figma MCP

The Figma MCP server connects Claude Code to your Figma workspace. It can read design files, describe component structure, extract design tokens (colors, fonts, spacing), and get screenshots of specific frames — all without you manually exporting anything.

Key operations:

  • get_design_context(nodeId, fileKey) — returns the full structure, code, and visual of a Figma frame.
  • get_screenshot(fileKey) — returns a visual snapshot of a frame.
  • get_metadata(fileKey) — returns file info and page names.

El servidor Figma MCP conecta Claude Code con tu espacio de trabajo de Figma. Puede leer archivos de diseño, describir la estructura de componentes, extraer design tokens (colores, fuentes, espaciado) y obtener capturas de pantalla de frames específicos — todo sin que vos tengas que exportar nada manualmente.

Operaciones clave:

  • get_design_context(nodeId, fileKey) — devuelve la estructura completa, código y visual de un frame de Figma.
  • get_screenshot(fileKey) — devuelve una captura visual de un frame.
  • get_metadata(fileKey) — devuelve información del archivo y nombres de páginas.
SQL Connection

Using Figma MCP is like running SELECT * FROM figma.design_tokens WHERE frame_id = :nodeId — you're querying Figma's database through a standardized interface, getting structured data back that you can use in code.

Conexión SQL

Usar Figma MCP es como ejecutar SELECT * FROM figma.design_tokens WHERE frame_id = :nodeId — estás consultando la base de datos de Figma a través de una interfaz estandarizada, obteniendo datos estructurados que podés usar en código.

2. Commanding Figma without designing2. Comandar Figma sin diseñar

You are not a Figma designer — and you don't need to be. Your role is to command Claude Code to interact with Figma programmatically. "Read this frame and tell me its colors." "Get the font used in this component." "Describe the layout of this design so I can replicate it in code."

This is the same skill you've been building all month: directing an AI agent through structured instructions rather than doing the work manually.

No sos diseñadora de Figma — y no necesitás serlo. Tu rol es comandar a Claude Code para que interactúe con Figma de manera programática. "Leé este frame y decime sus colores." "Obtené la fuente usada en este componente." "Describí el layout de este diseño para que pueda replicarlo en código."

Esta es la misma habilidad que venís desarrollando todo el mes: dirigir un agente de IA a través de instrucciones estructuradas en lugar de hacer el trabajo manualmente.

Exercise (~90 minutes)Ejercicio (~90 minutos)
  1. Open your Figma account (or create a free one at figma.com). You just need the free tier for this exercise.
    Abrí tu cuenta de Figma (o creá una gratis en figma.com). Solo necesitás el nivel gratuito para este ejercicio.
  2. Create a simple Figma file — add a rectangle, a text element with your name, and a color. This is your test frame. Keep it simple: this isn't about design, it's about reading data.
    Creá un archivo de Figma simple — agregá un rectángulo, un elemento de texto con tu nombre y un color. Este es tu frame de prueba. Que sea simple: esto no es sobre diseño, es sobre leer datos.
  3. Copy the file URL from Figma — it looks like https://www.figma.com/design/XXXX/filename?node-id=...
    Copiá la URL del archivo desde Figma — tiene esta forma: https://www.figma.com/design/XXXX/filename?node-id=...
  4. Ask Claude Code: "Using the Figma MCP server, read the design context from this URL: [paste your URL]. Describe what you see — colors, fonts, layout."
    Preguntale a Claude Code: "Usando el servidor Figma MCP, leé el contexto de diseño de esta URL: [pegá tu URL]. Describí lo que ves — colores, fuentes, layout."
  5. Ask Claude: "What are the exact color values used in that frame? Give them to me as CSS variables." — for example: --color-primary: #E87A53;
    Preguntale a Claude: "¿Cuáles son los valores exactos de color usados en ese frame? Dámelos como variables CSS." — por ejemplo: --color-primary: #E87A53;
  6. Ask Claude: "If I wanted to recreate this design in HTML/CSS, what would the code look like?" — review what Claude produces. Does it match what you see in Figma?
    Preguntale a Claude: "Si quisiera recrear este diseño en HTML/CSS, ¿cómo quedaría el código?" — revisá lo que produce Claude. ¿Coincide con lo que ves en Figma?
