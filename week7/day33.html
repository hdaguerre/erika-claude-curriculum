1. What image generation APIs do and how they work1. Qué hacen las APIs de generación de imágenes y cómo funcionan

Image generation models (like Replicate's Flux, Stability AI's Stable Diffusion, or Google's Imagen) take a text prompt and return a URL pointing to a generated image. You describe what you want in words, the model renders it, and you download the result.

The API pattern: POST /predictions with {"input": {"prompt": "...", "width": 1024, "height": 1024}} → response includes a URL you can download.

The big practical difference from text APIs: you're not parsing JSON to extract text — you're downloading a binary file (the image) from the returned URL.

Los modelos de generación de imágenes (como Flux de Replicate, Stable Diffusion de Stability AI, o Imagen de Google) reciben un prompt de texto y devuelven una URL que apunta a la imagen generada. Describís lo que querés en palabras, el modelo lo renderiza y vos descargás el resultado.

El patrón de la API: POST /predictions con {"input": {"prompt": "...", "width": 1024, "height": 1024}} → la respuesta incluye una URL que podés descargar.

La gran diferencia práctica con las APIs de texto: no estás parseando JSON para extraer texto — estás descargando un archivo binario (la imagen) desde la URL devuelta.

SQL Connection

Calling an image generation API is like calling a stored procedure that creates a file and returns its path: CALL generate_image(:prompt) RETURNING :image_url — you pass the description, get back a location.

Conexión SQL

Llamar a una API de generación de imágenes es como llamar a un stored procedure que crea un archivo y devuelve su ruta: CALL generate_image(:prompt) RETURNING :image_url — pasás la descripción, obtenés de vuelta una ubicación.

2. Building tools/generate_image.py2. Construyendo tools/generate_image.py

The tool takes a prompt string as input, calls the Replicate API, saves the downloaded image to an output/ folder, and returns the local file path.

Prompt engineering for images is its own skill — be specific about style, mood, composition, and aspect ratio:

  • Vague: "LinkedIn banner" → generic result
  • Specific: "Professional LinkedIn banner photo, clean minimalist design, warm terracotta and cream tones, abstract geometric shapes, 1584x396 pixels" → much better result

La herramienta recibe un string de prompt como entrada, llama a la API de Replicate, guarda la imagen descargada en una carpeta output/ y devuelve la ruta del archivo local.

El prompt engineering para imágenes es una habilidad propia — sé específica sobre el estilo, el mood, la composición y la relación de aspecto:

  • Vago: "banner de LinkedIn" → resultado genérico
  • Específico: "Banner de LinkedIn profesional, diseño minimalista limpio, tonos cálidos terracota y crema, formas geométricas abstractas, 1584x396 píxeles" → resultado mucho mejor
