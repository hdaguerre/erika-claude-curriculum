45 minutes — the big test

Run the complete workflow end-to-end Ejecutá el workflow completo de punta a punta

Open Claude Code in your mi-taller-code project. Then send this exact prompt:

"I have a photo of my abuela's alfajores I just made. Generate an Instagram caption in Spanish for this post."

Watch what happens. Ask yourself:

  • Did Claude read the CLAUDE.md context?
  • Did it follow the instagram_caption workflow?
  • Did the bakery-caption skill trigger?
  • Does the output sound like you — warm, personal, handmade — or does it sound generic?

If something didn't work as expected, that's not failure — that's exactly what this test is for. Adjust the piece that failed and run it again. Each iteration makes the system stronger.

Abrí Claude Code en tu proyecto mi-taller-code. Después enviá este prompt exacto:

"Tengo una foto de los alfajores de mi abuela que acabo de hacer. Generá una caption de Instagram en español para esta publicación."

Observá lo que pasa. Preguntate:

  • ¿Leyó Claude el contexto del CLAUDE.md?
  • ¿Siguió el workflow de instagram_caption?
  • ¿Se activó la habilidad bakery-caption?
  • ¿El resultado suena a vos — cálido, personal, artesanal — o suena genérico?

Si algo no funcionó como esperabas, eso no es un fracaso — es exactamente para lo que sirve esta prueba. Ajustá la pieza que falló y ejecutalo de nuevo. Cada iteración hace el sistema más robusto.
