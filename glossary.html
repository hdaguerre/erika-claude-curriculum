A plain-text file that stores secret credentials — API keys, passwords, tokens — outside your code. Python scripts load it using the python-dotenv library. Never commit this file to git; add it to .gitignore.

Un archivo de texto plano que almacena credenciales secretas — claves API, contraseñas, tokens — fuera de tu código. Los scripts de Python lo cargan usando python-dotenv. Nunca subas este archivo a git; agregalo al .gitignore.

Deterministic vs Non-deterministicDeterminista vs No Determinista

Deterministic: same input always produces the same output — Python scripts, SQL queries. Non-deterministic: output varies each run — LLMs like Claude. The WAT framework separates these on purpose: Claude handles decisions (non-deterministic); Python tools handle execution (deterministic).

Determinista: la misma entrada siempre produce la misma salida — scripts Python, consultas SQL. No determinista: la salida varía en cada ejecución — LLMs como Claude. El framework WAT los separa a propósito: Claude maneja las decisiones; las herramientas Python manejan la ejecución.

El formato universal para el intercambio de datos en APIs. Se parece a un diccionario de Python: {"name": "Acme", "domain": "acme.com"}. Los arrays se ven como listas: [{...}, {...}]. Si podés leer un result set de SQL, podés leer JSON.

TechnicalTécnico

JWTJWT

JSON Web Token. A signed, self-contained token an app issues after login to prove "this user is authenticated." Passed in request headers as Bearer [token]. Unlike API keys (static), JWTs expire and carry user identity. Common in OAuth flows.

JSON Web Token. Un token firmado y autocontenido que una app emite tras el login para probar "este usuario está autenticado". Se pasa en los headers como Bearer [token]. A diferencia de las claves API (estáticas), los JWTs expiran y llevan identidad del usuario. Común en flujos OAuth.

Un parámetro (0–1) que controla qué tan aleatorias son las respuestas del modelo. Temperatura baja (0–0.3) = más consistente, predecible — bueno para tareas estructuradas como extracción de datos. Temperatura alta (0.7–1) = más creativo y variado — bueno para brainstorming o copywriting.

TokenToken (LLM)

The basic unit an LLM uses to process text — roughly 4 characters or ¾ of a word. Every message and every response costs tokens. Context window size and API pricing are both measured in tokens. Not the same as an API key, which is also sometimes called a token.

La unidad básica que un LLM usa para procesar texto — aproximadamente 4 caracteres o ¾ de una palabra. Cada mensaje y cada respuesta cuesta tokens. El tamaño de la ventana de contexto y el precio de la API se miden en tokens. No es lo mismo que una clave API, que a veces también se llama token.

The ability for an LLM to call external functions or APIs — a calculator, a search engine, a database query, a Python script. Instead of guessing, the model decides when to call a tool and passes structured arguments. In the WAT framework, Tools (Python scripts) are what Claude calls to take real-world actions.

La capacidad de un LLM de llamar a funciones o APIs externas — una calculadora, un motor de búsqueda, una consulta a base de datos, un script de Python. En lugar de adivinar, el modelo decide cuándo llamar una herramienta y pasa argumentos estructurados. En el framework WAT, las Tools (scripts de Python) son lo que Claude llama para actuar en el mundo real.

Training CutoffFecha de Corte de Entrenamiento

The date after which an LLM has no knowledge of world events — its training data stops there. Anything after that (new laws, company news, recent pricing) it simply doesn't know. Fix: use the Perplexity API or inject current context directly into the prompt.

La fecha después de la cual un LLM no tiene conocimiento de eventos mundiales — sus datos de entrenamiento terminan ahí. Todo lo posterior (nuevas leyes, noticias de empresas, precios recientes) simplemente no lo sabe. Solución: usar la API de Perplexity o inyectar el contexto actual directamente en el prompt.

Workflows + Agents + Tools. Arquitectura de tres capas: los Workflows son SOPs en Markdown (instrucciones), el Agente es Claude (razonamiento), las Tools son scripts de Python (ejecución). Separar las responsabilidades hace al sistema confiable a escala.

WebhookWebhook

A reverse API call — instead of you calling a service, the service calls your URL when something happens. Example: when a contact replies to an Apollo sequence, Apollo POSTs to your webhook URL. Like a database trigger that fires an outbound HTTP call.

Una llamada API inversa — en lugar de que vos llamés a un servicio, el servicio llama a tu URL cuando algo sucede. Ejemplo: cuando un contacto responde a una secuencia de Apollo, Apollo hace un POST a tu webhook. Como un trigger de base de datos que dispara una llamada HTTP saliente.
