Setting Up Claude Code & Running server.py

Estimated time: ~2.5 hours
ConceptsConceptos

1. Installing Claude Code1. Instalando Claude Code

Claude Code is installed as an npm package. npm is a package manager — a tool for installing software from the command line. It comes bundled with Node.js.

The install command is: npm install -g @anthropic-ai/claude-code

The -g flag means global — install it once and it's available everywhere on your computer, not just in one specific folder. Without -g, the package only works in the current project directory.

Claude Code se instala como un paquete npm — npm es un gestor de paquetes para JavaScript/Node.js. Instalarlo es un solo comando: npm install -g @anthropic-ai/claude-code. El flag -g significa 'global' — instalarlo para que esté disponible en cualquier lugar, no solo en una carpeta. Una vez instalado, escribís claude en cualquier terminal para iniciarlo. Se conecta a tu cuenta de Anthropic.

SQL Connection

Installing npm packages is like installing Oracle client libraries — you run an installer once, it sets up the software, and from then on it's available in your environment. The -g flag is like installing to the system PATH rather than a local folder.

Conexión SQL

Instalar paquetes npm = instalar librerías cliente de Oracle. Ejecutar claude = abrir una sesión de SQL*Plus. Ejecutar python server.py = ejecutar un script de procedimiento almacenado desde la línea de comandos.

2. Running Claude Code2. Ejecutando Claude Code

Once installed, type claude in any terminal to start a session. Claude Code reads the files in your current working directory and connects to your Anthropic account. You type commands or questions, and it can read files, create files, run scripts, and help you build.

The experience is like pair programming at the command line — Claude sees your whole project and can act on it directly.

SQL Connection

Running claude is like opening a SQL*Plus session. You connect, you have context (the project folder instead of a database schema), and you can issue commands that get executed immediately.

3. Running server.py — your first script execution3. Ejecutando server.py — tu primera ejecución de script

Today you'll run python server.py from the curriculum/web/ folder. Let's be precise about what happens when you do this:

  1. Windows finds the Python interpreter on your system
  2. Python reads server.py line by line
  3. The script changes directory to its own folder
  4. It starts a simple web server on port 8080
  5. It opens your default browser to http://localhost:8080
  6. You see your curriculum dashboard in the browser

This is exactly how every "tool" in the WAT framework works — a Python script that does one specific, reliable thing when you run it.

Hoy vas a ejecutar tu primer script. Cuando corrés python server.py desde la carpeta curriculum/web/, Python lee el archivo server.py, lo ejecuta línea por línea, inicia un servidor web y abre el navegador. Esto es exactamente cómo funciona una 'herramienta' en el framework WAT — un script de Python que hace algo determinístico. Acabás de ejecutar un script. Eso es todo. Eso es lo que hacen todas las herramientas en este proyecto.

Key Insight

You just ran a script. That's it. That's all tools are — scripts that do one thing reliably. There's no magic. You type python server.py, Python executes it deterministically. This is what makes tools the most reliable layer in any AI workflow.

Idea Clave

Ejecutaste un script. Eso es todo. Eso es lo que hacen todas las herramientas en este proyecto: scripts que ejecutás desde la terminal que hacen una cosa específica de manera confiable.

Exercise (~60 minutes)Ejercicio (~60 min)
  1. Open VS Code and open the integrated terminal (Ctrl+`). Check if Node.js is installed: type node --version and press Enter. You should see something like v20.x.x. If you get an error, download Node.js from nodejs.org — choose the LTS version.
    Abrí VS Code y la terminal integrada (Ctrl+`). Verificá si Node.js está instalado: escribí node --version y presioná Enter. Deberías ver algo como v20.x.x. Si obtenés un error, descargá Node.js desde nodejs.org — elegí la versión LTS.
  2. Install Claude Code globally: npm install -g @anthropic-ai/claude-code. This will take a minute or two. You'll see progress output in the terminal.
    Instalá Claude Code globalmente: npm install -g @anthropic-ai/claude-code. Esto va a tardar uno o dos minutos. Vas a ver el progreso en la terminal.
  3. Confirm the installation: claude --version. You should see a version number. If you get "command not found," close and reopen VS Code to refresh the terminal environment.
    Confirmá la instalación: claude --version. Deberías ver un número de versión. Si obtenés "command not found", cerrá y volvé a abrir VS Code para actualizar el entorno de la terminal.
  4. Navigate to the curriculum web folder. In the terminal type: cd curriculum\web (Windows) or cd curriculum/web (Mac). Confirm with dir — you should see server.py in the file list.
    Navegá a la carpeta web del curriculum. En la terminal escribí: cd curriculum\web (Windows) o cd curriculum/web (Mac). Confirmá con dir — deberías ver server.py en la lista de archivos.
  5. Run the server: python server.py. Your browser should open automatically to http://localhost:8080. You'll see your curriculum dashboard. That's your own web app, served from your own computer.
    Ejecutá el servidor: python server.py. Tu navegador debería abrirse automáticamente en http://localhost:8080. Vas a ver tu dashboard del curriculum. Esa es tu propia aplicación web, servida desde tu propia computadora.
  6. Click around the dashboard. Click on Day 11 — mark it complete if you haven't yet. Watch the progress bar update. This is working software you're running locally.
    Explorá el dashboard. Hacé clic en el Día 11 — marcalo como completo si todavía no lo hiciste. Observá cómo se actualiza la barra de progreso. Este es software funcionando que estás ejecutando de manera local.
  7. When done, stop the server: press Ctrl+C in the terminal. The server stops. The browser page will no longer load. You can start it again any time with python server.py.
    Cuando termines, detené el servidor: presioná Ctrl+C en la terminal. El servidor se detiene. La página del navegador ya no va a cargar. Podés iniciarlo de nuevo en cualquier momento con python server.py.
